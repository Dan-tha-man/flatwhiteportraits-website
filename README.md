# Flat White Portraits

Marketing website for Flat White Portraits, a luxury photobooth and portrait experience service for weddings and events in Southern California.

## About

Flat White Portraits offers a premium alternative to traditional photobooths, featuring professional photographers, studio-quality lighting, and magazine-style portraits. Services include on-site portrait sessions, professional editing, digital galleries, and a digital guest book where guests can sign their photos with personal messages.

## Tech Stack

- HTML, CSS, JavaScript (no frameworks)
- Jekyll for GitHub Pages deployment
- HoneyBook integration for contact form
